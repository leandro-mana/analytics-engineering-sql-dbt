# Analytics Engineering with SQL and dbt — Notes

Study notes and hands-on exercises based on *Analytics Engineering with SQL and dbt* by Rui Machado & Hélder Russa (O'Reilly, 2023).

## Requirements

- [Python 3.12+](https://www.python.org/downloads/)
- [Poetry](https://python-poetry.org/docs/#installation)
- [Docker](https://docs.docker.com/get-docker/)

## Quick Start

```bash
make install
make up
```

## Book Reference

Place the book PDF in `docs/` (gitignored):

```bash
cp ~/path/to/"Analytics Engineering with SQL and dbt (2023).pdf" docs/
```
